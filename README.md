Este repositório documenta minha jornada estruturada de modernização arquitetural, evolução técnica e preparação para atuação global como Engenheiro de Software e Arquiteto de Sistemas. O PCI é um roadmap público, transparente e progressivo, cobrindo APIs modernas, Java avançado, DevOps, Cloud e práticas de engenharia utilizadas no mercado internacional.

---

## 🎯 **Objetivos do PCI**
- Migrar competências de sistemas legados para arquiteturas distribuídas modernas  
- Consolidar práticas de engenharia de software de nível global  
- Construir portfólio público com projetos reais  
- Criar base sólida para atuação internacional (2026–2028)  
- Documentar a evolução técnica de forma aberta e replicável
